### [Virtual File System](https://github.com/nirajnale/virtual-file-system)
UNIX-like file system implemented in C++ from scratch.
Implements a superblock, inode table, User File Descriptor Table (UFDT), and file
allocation blocks. System calls implemented: `Create`, `Open`, `Close`, `Read`,
`Write`, `LSeek`, `Stat`, `Fstat`, `Truncate`, `Rm`, `Ls`.
Supports file permissions (read/write), reference counting, link counts, and
multiple file descriptor types (regular file, directory).

`C++` `Systems Programming` `OS Internals` `Memory Management` `File Systems`

### [Generalized Data Structures Library](https://github.com/nirajnale/data-structures-library-cpp)
Type-agnostic C++ template library implementing four linked list variants:
Singly Linear, Singly Circular, Doubly Linear, and Doubly Circular.
Each structure supports insert and delete operations at first node, last node,
and arbitrary position, with full traversal. Built with templates so the same
implementation works for any data type without duplication.

`C++` `Templates` `OOP` `Data Structures` `Memory Management`

### [ProductHunt Scraper Suite](https://github.com/nirajnale/producthunt-scraper)

Stage 2 extracts detailed product information and maker profiles.

Each stage produces a structured CSV consumed by the next.
Outputs 500+ product records and 200+ enriched maker profiles per run.
